Third-Party Vendor Risk Assessment (GRC Project)

This project focuses on evaluating security risks associated with third-party vendors such as cloud providers, payment gateways, and email services.

Key Activities:
- Vendor identification and classification
- Risk assessment using likelihood and impact
- Vendor security questionnaire development
- Risk scoring and prioritization
- Security recommendations

Vendors Assessed:
- Cloud Provider (High Risk)
- Payment Gateway (High Risk)
- Email Service (Medium Risk)

Tools Used:
- Microsoft Word
- Microsoft Excel

This project demonstrates practical understanding of vendor risk management in GRC.
